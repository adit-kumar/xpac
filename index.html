xpac

A Cross-platform Package Manager

xpac is an idea that four Computer Science students at Purdue University came up with the aim of providing a unified packaging solution to users.

As an idea that originated in the minds of one of our team members, and taken further as a capstone project for CS 30700: Software Engineering at Purdue, we aim to take it even further through the contributions of developers all over the globe.

Installation Instructions

Follow the steps in the order given:

1. In your terminal, run git clone https://github.com/parth-shel/xpac.git -b prod xpac .

2. Open the directory by running cd xpac .

3. Start installation by running sudo make install .

4. Enter your password when prompted.

You are all set!
