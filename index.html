xpac

A Cross-platform Package Manager

xpac is our aim of providing a unified packaging solution to users across a multitude of platforms.

Installation Instructions

Follow the steps in the order given:

1. In your terminal, run git clone https://github.com/parth-shel/xpac.git -b prod xpac.

2. Open the directory by running cd xpac.

3. Start installation by running make.

4. Complete installation by running sudo make install.

5. Enter your password when prompted.

You are all set!
